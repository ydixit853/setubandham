# Setubandham (सेतुबंधम) 🌉
*Built with ❤️ by Yash Dixit for the AI Hackathon*

## 🚀 Overview
Setubandham is an AI-powered, bilingual hyperlocal issue solver built for the citizens and municipalities of Madhya Pradesh.

## 🔐 Evaluation Testing Credentials
To experience the full agentic depth of the application, please test both portals:
**1. Citizen Portal:** Browse the map and report an issue.
**2. Official Portal:** 
- Username: `officer_parasia`
- Password: `mp_admin@2026`
*(This logs you into the Chhindwara/Parasia municipal dashboard to review and resolve the citizen reports).*

## 🛠 Tech Stack
- Google AI Studio & Gemini 2.5 Flash (Agentic Logic & Vision)
- Google Maps & Geocoding API (Location Services)
- Google Cloud (Deployment)
